## Quick Start

```bash
# Clone the repository
git clone https://github.com/CodeMasters-AiFactory/Stargate.git

# Install dependencies
cd Stargate
npm install

# Start development server
npm run dev
```

Visit `http://localhost:5000` to see Stargate Portal in action!
