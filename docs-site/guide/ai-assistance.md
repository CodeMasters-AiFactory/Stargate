# AI Assistance

Stargate Portal is powered by advanced AI to help you build better websites.

## AI Capabilities

### Content Generation
- Write compelling headlines and copy
- Generate product descriptions
- Create blog post drafts
- Craft call-to-action text

### Design Assistance
- Suggest color palettes
- Recommend layouts
- Optimize typography
- Improve visual hierarchy

### Image Creation
- Generate custom hero images
- Create product photography
- Design illustrations
- Make icons and graphics

### Optimization
- SEO recommendations
- Accessibility improvements
- Performance suggestions
- Mobile optimization

## How to Use AI

### Method 1: Merlin Chat

Open the Merlin sidebar and type natural language requests:

> "Add a testimonials section with 3 customer reviews"
> "Change the primary color to navy blue"
> "Write a compelling headline for a law firm"

### Method 2: Inline Suggestions

Click the AI icon next to any text field:
- Headlines get title suggestions
- Paragraphs get content variations
- Buttons get CTA options

### Method 3: Generate Button

Click **Generate** on empty sections:
- AI fills placeholder content
- Matches your brand voice
- Uses context from your project

### Method 4: Improve Selection

Select any element and click **Improve**:
- AI enhances the content
- Suggests alternatives
- Fixes grammar/spelling

## AI Best Practices

### Provide Context

The more context you give, the better results:

**Vague:** "Write a headline"
**Better:** "Write a headline for a luxury real estate agency targeting high-net-worth buyers in Cape Town"

### Iterate

Don't settle for the first result:
1. Generate initial content
2. Ask for variations
3. Combine the best parts
4. Refine manually

### Review Everything

AI suggestions should be reviewed:
- Check factual accuracy
- Verify brand voice
- Ensure legal compliance
- Test for accessibility

## AI Limitations

- Cannot access external websites
- May not know recent events
- Results vary in quality
- Needs human review

## API Keys Required

For AI features, configure these in `.env`:

```env
ANTHROPIC_API_KEY=your-claude-key
LEONARDO_AI_API_KEY=your-leonardo-key
GOOGLE_SEARCH_API_KEY=your-google-key
```
