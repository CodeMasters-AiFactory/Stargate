## What Makes a World-Class Website

A world-class website is not just a template that renders. It is a carefully crafted digital experience that:

### 1. Strong Brand & Visual System
- **Cohesive colour palette** with clear primary, secondary, and accent colours
- **Professional typography** with strong hierarchy (display fonts for headings, readable body fonts)
- **Consistent spacing system** (8px, 16px, 24px, 32px, 48px, 64px, 96px grid)
- **Unified layout patterns** across all pages
- **Visual identity** that reflects the business personality and industry standards
- **No generic stock photos** - use custom images, illustrations, or high-quality curated visuals

### 2. Clear Information Architecture & User Journeys
- **Logical site structure** that matches user mental models
- **Clear navigation** with intuitive labels
- **User journeys** designed for each audience type (e.g., businesses vs. individuals)
- **Breadcrumb navigation** for multi-level sites
- **Search functionality** for content-rich sites
- **Mobile-first responsive design** that works flawlessly on all devices

### 3. Deep, Specific Content (Not Generic Filler)
- **Industry-specific language** that demonstrates expertise
- **Detailed service descriptions** with specific use cases
- **Location-specific content** (not generic "we serve clients")
- **Real value propositions** (not "we are the best")
- **Case studies or examples** (even anonymised) that show real work
- **FAQ sections** that answer actual questions
- **No placeholder text** or "lorem ipsum" in production

### 4. Strong Conversion & Trust Elements
- **Clear, action-oriented CTAs** (not generic "Learn More")
- **Multiple conversion points** throughout the user journey
- **Social proof** (testimonials, case studies, client logos, awards)
- **Trust signals** (certifications, memberships, years in business)
- **Easy contact options** (phone, email, form, chat)
- **Transparent pricing or fee structure** where appropriate
- **Privacy and security indicators** (SSL, GDPR compliance)

### 5. SEO Best Practices
- **Keyword-rich, unique page titles** (50-60 characters, includes location/service)
- **Compelling meta descriptions** (150-165 characters, includes CTA)
- **One H1 per page** with primary keyword
- **Proper heading hierarchy** (H1 â†’ H2 â†’ H3)
- **Internal linking structure** between related pages
- **Schema markup** (Organization, LocalBusiness, Service, FAQPage)
- **Alt text on all images**
- **Semantic HTML5** structure
- **Fast page load times** (< 3 seconds)
- **Mobile-friendly** (responsive, touch-friendly)

### 6. Accessibility & Performance
- **WCAG 2.1 AA compliance** (minimum)
- **Keyboard navigation** support
- **Screen reader compatibility**
- **Color contrast ratios** (4.5:1 for text, 3:1 for UI)
- **Skip links** for keyboard users
- **ARIA labels** where needed
- **Performance optimization** (lazy loading, image optimization, code splitting)
- **Core Web Vitals** (LCP < 2.5s, FID < 100ms, CLS < 0.1)

### 7. Creativity & Differentiation
- **Unique visual identity** (not a generic template)
- **Innovative layout solutions** that serve the content
- **Thoughtful micro-interactions** and animations
- **Custom illustrations or photography** (not stock photos)
- **Memorable brand elements** that differentiate from competitors
- **Industry-appropriate creativity** (professional but not boring)

## Rating Checklist (0-10 Scale)

Rate each website in these categories:

### Visual Design & Layout (0-10)
- **0-3:** Poor - Inconsistent, unprofessional, cluttered, no visual hierarchy
- **4-5:** OK - Basic styling, some consistency, but generic template feel
- **6-7:** Good - Professional appearance, consistent branding, good spacing
- **7.5-8.5:** Excellent - Strong visual identity, polished design, industry-leading aesthetics
- **8.5-10:** World-class - Exceptional design, memorable, sets new standards

**Key Questions:**
- Does it look like a $50,000+ agency build?
- Is the colour palette professional and cohesive?
- Is typography hierarchy clear and readable?
- Are images custom or high-quality curated (not generic stock)?
- Is spacing consistent and generous?

### UX & Structure (0-10)
- **0-3:** Poor - Confusing navigation, broken links, poor mobile experience
- **4-5:** OK - Basic navigation works, but user journey unclear
- **6-7:** Good - Clear structure, easy to navigate, responsive
- **7.5-8.5:** Excellent - Intuitive flows, multiple user journeys, excellent mobile UX
- **8.5-10:** World-class - Seamless experience, innovative navigation, perfect information architecture

**Key Questions:**
- Can users find what they need in 3 clicks or less?
- Is navigation intuitive and consistent?
- Are user journeys designed for different audience types?
- Does mobile experience match desktop quality?
- Is the site structure logical and scalable?

### Content & Positioning (0-10)
- **0-3:** Poor - Generic filler text, no specific information, placeholder content
- **4-5:** OK - Some specific content, but still generic in places
- **6-7:** Good - Specific services described, location mentioned, clear value props
- **7.5-8.5:** Excellent - Deep, specific content, industry expertise shown, compelling positioning
- **8.5-10:** World-class - Thought leadership content, unique positioning, exceptional clarity

**Key Questions:**
- Is content specific to the business (not generic)?
- Are services described in detail with use cases?
- Is location-specific content included?
- Are value propositions clear and differentiated?
- Is there social proof (case studies, testimonials)?

### Conversion & Trust (0-10)
- **0-3:** Poor - No clear CTAs, no trust signals, hard to contact
- **4-5:** OK - Basic contact form, some CTAs, minimal trust elements
- **6-7:** Good - Clear CTAs, contact options, some social proof
- **7.5-8.5:** Excellent - Multiple conversion points, strong trust signals, easy contact
- **8.5-10:** World-class - Optimized conversion funnel, exceptional trust building, seamless contact

**Key Questions:**
- Are CTAs clear, action-oriented, and prominent?
- Are there multiple conversion opportunities?
- Is social proof present and credible?
- Are trust signals visible (certifications, memberships, years)?
- Is contact information easy to find and use?

### SEO Foundations (0-10)
- **0-3:** Poor - No SEO optimization, generic titles, no structure
- **4-5:** OK - Basic titles, some headings, minimal optimization
- **6-7:** Good - Keyword-rich titles, proper headings, basic schema
- **7.5-8.5:** Excellent - Comprehensive SEO, schema markup, internal linking
- **8.5-10:** World-class - Advanced SEO, technical excellence, content strategy

**Key Questions:**
- Are page titles unique, keyword-rich, and include location?
- Is heading hierarchy proper (H1 â†’ H2 â†’ H3)?
- Is schema markup implemented?
- Are internal links structured for SEO?
- Are images optimized with alt text?
- Is the site fast and mobile-friendly?

### Creativity & Differentiation (0-10)
- **0-3:** Poor - Generic template, no unique elements, looks like every other site
- **4-5:** OK - Some customization, but still template-like
- **6-7:** Good - Custom design elements, unique layout, some creativity
- **7.5-8.5:** Excellent - Distinctive visual identity, innovative solutions, memorable
- **8.5-10:** World-class - Exceptional creativity, industry-leading innovation, unforgettable

**Key Questions:**
- Does it stand out from competitors?
- Are there unique design elements?
- Is the visual identity memorable?
- Are custom images/illustrations used?
- Does it feel like a custom build (not a template)?

## Rating Interpretation

Calculate the average score across all 6 categories:

- **Poor:** Average < 4.0
  - Site has fundamental issues. Not production-ready.
  - Example: Generic template, placeholder content, broken layout, no real brand identity.

- **OK:** Average 4.0 - 5.9
  - Basic functionality works, but needs significant improvement.
  - Example: Basic template that renders, but generic content, weak CTAs, minimal SEO.

- **Good:** Average 6.0 - 7.4
  - Professional quality, but not exceptional. Some areas need work.
  - Example: Clean design, some specific content, but still feels template-like, missing trust elements.

- **Excellent:** Average 7.5 - 8.4
  - High-quality site that competes well in its industry.
  - **CRITICAL REQUIREMENT: No single category below 7.5/10**
  - Must have: Strong brand identity, specific content, clear user journeys, strong CTAs, good SEO, some differentiation.

- **World-Class:** Average > 8.5
  - Exceptional quality that sets industry standards.
  - **CRITICAL REQUIREMENT: All categories â‰¥ 8.5/10, AND no category below 7.5/10**
  - Must have: Memorable brand, exceptional content depth, innovative UX, strong conversion optimization, advanced SEO, unique creative elements.

## Critical Rules for Rating

1. **Never call a site "excellent" if ANY category is below 7.5/10**
2. **Compare against real, top-tier websites in that industry** (not just "it works" or "it doesn't crash")
3. **Be specific in feedback** - "add location keywords to H1" not "improve SEO"
4. **Consider the target audience** - a law firm needs different standards than a portfolio site
5. **Evaluate actual content quality** - not just "content exists"
   - Generic filler like "We deliver exceptional quality" = poor content
   - Specific details like "Corporate law for SMEs in Pretoria" = good content
6. **Check for generic elements** - stock photos, template layouts, placeholder text, confetti backgrounds
7. **Test user journeys** - can users actually accomplish their goals?
8. **Verify technical quality** - SEO, accessibility, performance matter
9. **Be honest and critical** - if it looks like a generic template, rate it as such (2-3/10), not "excellent"
10. **Require brand identity** - colors, typography, spacing must feel like a unique brand, not arbitrary choices

## Examples of What "Excellent" Means

### Law Firm Website (Excellent = 7.5-8.5)
- **Visual Design:** Professional colour scheme (navy/blue), clean typography, custom hero image, consistent spacing
- **UX:** Clear navigation (Services, About, Contact), easy to find practice areas, mobile-optimized
- **Content:** Specific service descriptions, location mentioned, case studies, FAQ section
- **Conversion:** "Book Consultation" CTAs, phone number prominent, contact form, trust badges
- **SEO:** "Law Firm in [CITY]" in titles, proper headings, schema markup, fast loading
- **Creativity:** Unique layout, custom imagery, professional but memorable design

### NOT Excellent (What We've Been Building)
- Generic template layout
- Stock photos
- Placeholder text
- Generic CTAs ("Learn More")
- No location-specific content
- Basic styling without brand identity
- No social proof
- Minimal SEO optimization
