## üî¨ CALCULATION METHODOLOGY

The local analyzer uses **rules-based scoring** with the following methodology:

### Scoring System Overview

Each category starts with a **base score** and gains/loses points based on specific criteria:

1. **Visual Design & Layout** (Base: 5.0)
   - +1.0: Cohesive color scheme (‚â•2 colors detected)
   - +0.5: Custom images (not stock photo sites)
   - +0.5: No confetti/playful elements
   - +0.5: CSS grid/flexbox detected
   - +1.0: World-class bonus (‚â•3 colors + custom images + no confetti)
   - **Max: 10.0**

2. **UX & Structure** (Base: 5.0)
   - +1.0: Navigation present
   - +1.0: Proper heading hierarchy (1 H1, ‚â•3 H2s)
   - +0.5: Mobile responsive indicators
   - +0.5: Multiple CTAs (‚â•3)
   - +0.5: Internal linking (‚â•10 links)
   - +1.0: World-class bonus (nav + 1 H1 + ‚â•5 H2s + ‚â•5 CTAs)
   - **Max: 10.0**

3. **Content & Positioning** (Base: 5.0)
   - -2.0: **Penalty** for generic filler phrases
   - +1.0: Content depth (‚â•500 words)
   - +1.0: Deep content (‚â•1000 words)
   - +1.0: Location references present
   - +1.0: Specific examples (years, testimonials, case studies)
   - +1.0: World-class bonus (‚â•1500 words + no filler + location)
   - **Max: 10.0**

4. **Conversion & Trust** (Base: 3.0 - starts lower)
   - +2.0: Strong CTAs ("Book a Consultation", "Get Started", "Start Free Trial")
   - +0.5: Weak CTAs ("Contact Us", "Learn More")
   - +1.0: Phone number present
   - +1.0: Email address present
   - +0.5: Physical address present
   - +1.0: Contact form present
   - +1.0: Trust elements (testimonials, certifications, awards)
   - +1.0: World-class bonus (strong CTAs + all contact methods + form)
   - **Max: 10.0**

5. **SEO Foundations** (Base: 5.0)
   - +1.0: Title length (30-60 characters)
   - +0.5: Keyword-rich title (not generic "Home | Site")
   - +1.0: Meta description (120-165 characters)
   - +1.0: One H1 per page
   - +1.0: Schema markup present
   - +0.5: Descriptive headings (not generic)
   - +0.5: World-class bonus (title ‚â•40 chars + schema + 1 H1)
   - **Max: 10.0**

6. **Creativity & Differentiation** (Base: 5.0)
   - +0.5: Unique color scheme (‚â•3 colors)
   - +0.5: Custom images
   - +0.5: Brand story/narrative present
   - +0.5: Memorable tagline
   - +0.5: Not generic template
   - +1.0: World-class bonus (colors + images + story + not generic)
   - **Max: 10.0**

### Final Verdict Calculation

- **Poor:** Average < 4.0
- **OK:** Average 4.0-5.9
- **Good:** Average 6.0-7.4 OR average ‚â•7.5 but any category < 7.5
- **Excellent:** Average 7.5-8.4 AND **all categories ‚â• 7.5**
- **World-Class:** Average > 8.5 AND **all categories ‚â• 8.5**

**Critical Rule:** A site cannot be "Excellent" if ANY category is below 7.5/10.

## üîç METHODOLOGY LIMITATIONS

### What the Analyzer Detects:

‚úÖ HTML structure (headings, links, forms)  
‚úÖ Text content (word count, patterns)  
‚úÖ Meta tags (title, description)  
‚úÖ Schema markup  
‚úÖ Basic CSS indicators (grid, flexbox)  
‚úÖ Contact information patterns (phone, email regex)

### What the Analyzer Cannot Detect:

‚ùå CSS-based color schemes (if colors defined in external stylesheets)  
‚ùå JavaScript-rendered content (if content loaded dynamically)  
‚ùå Image quality/aesthetics (only checks if images exist and aren't stock sites)  
‚ùå Animation/micro-interactions  
‚ùå Mobile responsiveness (only checks for viewport meta tag)  
‚ùå Accessibility features  
‚ùå Performance metrics

### Why Some Scores May Seem Low:

- **Apple (5.0 Visual Design):** Colors likely in CSS, not HTML attributes
- **Tesla (5.5 UX):** Navigation may be JavaScript-rendered
- **IBM (4.0 Conversion):** Contact info may be in footer/JavaScript
